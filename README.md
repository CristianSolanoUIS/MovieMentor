# MovieMentor_Proyecto Automatas
Este proyecto busca encontrar una forma de recomendar películas mediante el uso de Automatas. En el repositorio se encuentra la presentación de la propuesta, el lenguaje formal y el diagrama de transiciones, la presentación del proyecto, el notebook del proyecto para ejecutarse en Google Colab y finalmente el video del proyecto. ¡Espero que sea de su agrado y logren encontrar lo que estaban necesitando para eliminar su aburrimiento!.
